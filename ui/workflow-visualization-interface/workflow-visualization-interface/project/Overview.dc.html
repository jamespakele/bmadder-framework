Two ways to see the pipeline run

The orchestrator drives stories through agents from a single Rust binary. Both directions below turn that headless CLI into a visual surface — they teach the cycle (batch) and iterative (story-by-story) workflows, show stories moving through states live, and expose the agents, routing and bmadder.toml config.
